# Kotlin_study
kotlin development env setting utils on bash

## usage

```shell
$ git clone https://github.com/ToshihitoKon/Kotlin_study.git
$ cd Kotlin_study
```

### kotlin compiler install

```shell
$ make install
$ source ~/.profile
```

### build

exec `make` or `make build` after edit main.kt

### running 

```shell
$ make run
```
